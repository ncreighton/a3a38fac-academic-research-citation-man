# Academic Research Citation Manager and Paper Tracker VS Code Extension

> A VS Code extension for academic researchers, PhD students, and technical writers managing literature reviews and citation workflows. Imports DOI references to auto-populate BibTeX entries, generates formatted citations in APA, MLA, IEEE, and Chicago styles, highlights uncited claims in manuscripts with inline warnings, tracks reading status of papers in a sidebar panel, and exports reference lists compatible with LaTeX, Word, and Zotero.

## Installation

### From VS Code Marketplace
1. Open VS Code
2. Press `Ctrl+P` (or `Cmd+P` on Mac)
3. Type `ext install digitalforge.a3a38fac`
4. Press Enter

### From VSIX file
1. Download the `.vsix` file
2. In VS Code: `Extensions > ··· > Install from VSIX`
3. Select the downloaded file

## Usage

Press `Ctrl+Shift+P` and search for `Academic Research Citation Manager and Paper Tracker VS Code Extension` to see all available commands.

**Keyboard shortcut:** `Ctrl+Shift+Alt+A` (or `Cmd+Shift+Alt+A` on Mac)

## Settings

| Setting | Default | Description |
|---------|---------|-------------|
| `a3a38fac.enabled` | `true` | Enable/disable the extension |
| `a3a38fac.autoRun` | `false` | Auto-run on file open |
| `a3a38fac.logLevel` | `"info"` | Log verbosity |
